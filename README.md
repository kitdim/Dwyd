# Dwyd
What is Dwyd? These are microservices that are programs for processing price data, parsing data, interacting with customers via Telegram bots, and much more. 

You can find more detailed information describing what each service does at the links provided in the following block.

### Links
1. [DwydServer](https://github.com/kitdim/Dwyd/tree/master/DwydServer)
2. [DwydBot](https://github.com/kitdim/Dwyd/tree/master/DwydBot)
